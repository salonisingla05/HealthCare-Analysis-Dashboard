# 🏥 Healthcare Analysis Dashboard

This project presents a **Power BI Healthcare Dashboard** that analyzes hospital financial and operational data, including billing, treatment costs, insurance coverage, and departmental performance.

## 📊 Key Metrics

* **Total Billing Amount:** £1M – £3M
* **Average Billing Amount:** ~£674 – £709
* **Medication Cost:** £212K – £546K
* **Treatment Cost:** £1M – £3M
* **Total Insurance Coverage:** £881K – £2.23M
* **Out-of-Pocket Expenses:** £504K – £1.13M
* **Room Charges:** £141K – £180K


## 🧠 Insights from Dashboard

### 💉 Procedure-wise Billing

* Highest revenue generated from:

  * X-Ray
  * CT Scan
  * MRI Scan
* Lower contribution from:

  * Ultrasound
  * Blood Tests

### 🏥 Department-wise Analysis

* **Orthopedics** contributes the highest (~40%)
* Followed by:

  * Cardiology
  * General Surgery
  * Neurology
  * Pediatrics


### 🩺 Diagnosis & Service Insights

* Key conditions analyzed:

  * Asthma
  * Hypertension
  * Fracture
  * Appendicitis
  * Migraine
* Service types show varied cost distribution across diagnoses



### 💰 Financial Overview

* Insurance covers a significant portion of expenses
* Patients still incur notable **out-of-pocket costs**
* Balanced distribution between treatment and medication expenses
